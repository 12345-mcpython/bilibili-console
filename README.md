# LBCC

## 介绍
Laosun Bilibili Console Client(LBCC) 是laosun写的Bilibili命令行客户端。

## 实现功能
----
- [ ] 登录。
   - [ ] 密码登录。
   - [ ] 二维码登录。
   - [ ] 手机验证码登录。
- [x] 推荐。
- [ ] 搜索。
   - [x] 视频搜索。
   - [ ] 其他搜索。
- [x] 播放视频。
   - [x] 通过地址播放视频。 
   - [x] 通过推荐播放视频。
   - [x] 通过视频搜索播放视频。
   - [x] 播放视频带弹幕。
   
- [x] 预览评论

- [ ] 互动能力。
   - [ ] 评论。
   - [x] 点赞。
   - [ ] 投币。
   - [ ] 转发。
   - [x] 三连。
   - [ ] 发送弹幕。     

## 使用

创建cookie.txt与collection.txt

把你账号(如果有的话)的cookie拷贝在cookie.txt里

安装mpv与python

输入pip install -r requirements.txt

输入python main.py即可运行
