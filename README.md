# LBCC

## 介绍
Laosun Bilibili Console Client(LBCC) 是laosun写的Bilibili命令行客户端。

## 实现功能
----
- [ ] 登录。
   - [x] 密码登录。
   - [ ] 二维码登录。
   - [x] 手机验证码登录。
- [x] 推荐。
- [ ] 搜索。
   - [x] 视频搜索。
   - [ ] 其他搜索。
- [x] 播放视频。
   - [x] 通过地址播放视频。 
   - [x] 通过推荐播放视频。
   - [x] 通过视频搜索播放视频。
   - [x] 播放视频带弹幕。
   
- [x] 预览评论

- [ ] 互动能力。
   - [ ] 评论。
   - [x] 点赞。
   - [ ] 投币。
   - [ ] 转发。
   - [x] 三连。
   - [ ] 发送弹幕。     

## 使用

1. pip install -r requirements.txt
2. 复制账号cookie(或python main.py 输入login登录)拷贝至cookie.txt
3. 创建collection.txt
4. python main.py
